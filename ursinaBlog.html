What is Ursina?

Ursina is a Library usable with Python. You can use the library to create 2D and 3D games. I chose Ursina For my 'Technical Blog' because of the simplicity. Creating characters, the playable environment, and functions for your game are easy to learn and master. I started learning on it during the last semester switch.

How to use Ursina?

Like all installs with python, open a terminal and input 'pip install ursina'

I already have Ursina installed, so instead of 'requirements already satisfied' you'll see downloads happening.

INSTALL CONTENTS

  • panda-3d, panda-3d-gltf & panda3d-simplepbr>=0.6
  • pillow
  • pyperclip
  • screeninfo

Important Imports

Just like importing random, time and sys with Python, Ursina needs to be imported. An absolute necessity is 'from ursina import *' which imports the whole Ursina library. If you like First Person games (like me) you can type 'from ursina.prefabs.first_person_controller import FirstPersonController'
